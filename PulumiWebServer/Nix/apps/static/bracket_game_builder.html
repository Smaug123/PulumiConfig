How to Build a Puzzle

  1. Paste or type your source text in the first text box
  2. Click "Start Building" to begin
  3. Select text in the working area to create a clue
  4. Enter your clue text and click "Replace"
  5. Continue adding clues (you can nest clues within clues!)
  6. Copy the JSON output when you're done
